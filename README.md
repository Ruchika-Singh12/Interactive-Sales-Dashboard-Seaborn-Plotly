# 📊 Interactive Sales Dashboard

> **Week 6 Internship Project – The Developers Arena**

## 📌 Project Overview

The objective was to create an interactive sales dashboard that analyzes sales performance using Python and presents business insights through attractive and interactive visualizations.

The dashboard helps understand sales trends, product performance, regional revenue, pricing distribution, and customer purchasing behavior using both static and interactive charts.

---

## 🎯 Objectives

- Analyze sales data efficiently.
- Generate meaningful business insights.
- Create professional data visualizations.
- Build interactive dashboards using Plotly.
- Demonstrate data analysis skills using Python.

---

## 🛠 Technologies Used

- Python 3.12
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook
- Visual Studio Code

## ▶️ How to Run

### Install Required Libraries

```bash
pip install -r requirements.txt
```

### Run Dashboard

```bash
python dashboard.py
```

### Open Interactive Dashboard

Open the generated HTML files inside the **visualizations** folder using any web browser.
